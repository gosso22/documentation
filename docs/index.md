# Welcome to FRI-LOG Documentation

This is the full documentation on how to use Fri-Log for broadcasters as well as Farm Radio Staff. If you dont understand anything written here please contact ICT Officer for a respective country.

## Commands

* `mkdocs new [dir-name]` - Create a new project.
* `mkdocs serve` - Start the live-reloading docs server.
* `mkdocs build` - Build the documentation site.
* `mkdocs help` - Print this help message.

## Project layout

    mkdocs.yml    # The configuration file.
    docs/
        index.md  # The documentation homepage.
